# Security Policy

## Supported Versions

Jurisflow is in active development. Security fixes are applied on the main development line first.

## Reporting A Vulnerability

Please do not report security vulnerabilities through public GitHub issues.

Instead:

1. Open a private GitHub security advisory for this repository if that feature is enabled.
2. If that is not available, contact the maintainer privately through GitHub: [@SebastianBoehler](https://github.com/SebastianBoehler).
3. Include a clear description, reproduction steps, affected areas, impact assessment, and any suggested mitigation.

You can expect:

- acknowledgement after initial review
- follow-up questions if reproduction details are incomplete
- coordination on disclosure timing when the report is valid

Please avoid public disclosure until a fix or mitigation is available.
